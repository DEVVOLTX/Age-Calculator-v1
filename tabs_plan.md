Tabs + Features Roadmap

## الهدف
إضافة بنية Tabs داخل الصفحة بحيث نستطيع إضافة:
1) حاسبة العمر الشخصية (الحالي)
2) حاسبة عمر الحيوانات الأليفة
3) الذكرى السنوية المميزة/فرق الوقت بين تاريخين
(4 لاحقًا: التقاعد/الإنتاجية)

## التسليم المطلوب في المرحلة الأولى
- Tabs يعملان UI/UX بدون مكتبات
- كل تبويب له منطق مستقل داخل script.js
- حفظ الثيم يبقى كما هو
- localStorage الخاص ببيانات الميلاد يبقى كما هو

## قرار التصميم
- إبقاء RTL على مستوى الصفحة
- استخدام classnames عامة: tab-buttons, tab-button, tab-panel

## الخطة التقنية (high level)
- index.html: تحويل المحتوى الحالي إلى tab-panel 1، وإضافة tab-panel 2 و 3
- script.js:
  - فصل دوال حساب العمر الحالية إلى calcPersonalAge()
  - إضافة calcPetAge() بناء على السنة الأولى/الثانية ثم معامل تقريبي حسب نوع الحيوان
  - إضافة calcAnniversaryDiff() باستخدام diffYMD وعداد الأيام/الساعات/الدقائق
- style.css: تصميم Tabs لتكون UI مناسبة مع شكل البطاقات الحالية

## ملاحظة
لن يتم تضمين Quotable API أو Chart.js في المرحلة الأولى حتى لا نضيف dependencies أو تعقيد.
